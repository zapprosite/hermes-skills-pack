# Refrimix Delivery Conventions

## Folder convention
Use a dated folder for each finished carousel bundle.

Suggested pattern:
- `/home/will/Imagens/carrosseis/YYYY-MM-DD-topic-slug/`

## Bundle contents
- Copy outline
- Visual direction notes
- Image prompts
- Caption
- Hashtags
- Final QA checklist

## Delivery rule
Prefer a ready-to-use bundle path in the final response so the user can open it directly.

## Session hygiene
- Keep the skill itself generic.
- Put topic-specific working notes here, not in SKILL.md.
- Avoid one-skill-per-session artifacts.
