ChatGPT Plus dashboard notes for Instagram production

Use the dashboard as a production studio when it reduces Hermes API usage.

Best-fit surfaces from this session:
- Projects: campaign workspace for one brand/theme (briefs, drafts, reference links, asset organization).
- GPTs: repeatable assistants for copy critique, CTA tuning, caption variants, and brand-voice checks.
- Images: generate and iterate on visual concepts, covers, and carousel art directions.
- Canvas: polish copy/layout collaboratively; useful for slide text, hooks, captions, and alternative phrasings.

Operational pattern:
1) Keep one Project per campaign/theme.
2) Use GPTs for repeatable review passes, not one-off ideation only.
3) Move visual exploration into Images before final production.
4) Use Canvas to refine the exact copy that will ship.

Session-specific browser/login workflow:
- Prefer Chrome CDP for persistent manual login sessions.
- Use Camofox/VNC when the automated browser must keep a stable profile across runs.
- Reuse the same userId/sessionKey for persistence.

Content output target from this workflow:
- editorial, architectural, premium Instagram assets
- carousels and stories that feel authored rather than “template-generated”
- short-form content that can be repurposed into reels and captions
