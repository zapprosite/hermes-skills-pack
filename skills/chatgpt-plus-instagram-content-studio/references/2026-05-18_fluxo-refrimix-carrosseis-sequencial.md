## O aprendizado central

O maior erro do fluxo anterior era tratar a geração visual como primeiro problema. O carrossel perfeito nasceu porque a conversa resolveu ANTES:

1. tema;
2. audiência;
3. objetivo;
4. copy de todas as páginas;
5. título curto de cada página;
6. subtítulo/subtexto de cada página;
7. direção visual de cada página;
8. prompt de Image-2 por página;
9. só depois: geração sequencial das imagens.

Ou seja: Image-2 deve receber uma página que já sabe o que precisa dizer. Não gerar fundo bonito e depois descobrir a copy.

## Estrutura vencedora observada

Usar carrossel de 5 páginas:

1. Página 1 — Hook forte
   - Para interromper o scroll.
   - Fórmula: `O erro mais caro em [tema] começa antes de [ação óbvia].`

2. Página 2 — Virada de crença
   - Mostra que o problema real não é óbvio.
   - Fórmula: `[Ação comum] sem [planejamento técnico] é [risco].`

3. Página 3 — Consequências
   - Mostra perdas/riscos em bullets curtos.
   - Usar poucos itens, sem bloco de texto pesado.

4. Página 4 — Solução técnica
   - Posiciona Refrimix como especialista técnico premium.
   - Integra arquitetura + climatização + infraestrutura + automação.

5. Página 5 — CTA premium
   - Convite comercial sutil: projeto, análise ou consultoria.
   - Sem urgência barata.

## Tipografia vencedora

A conversa perfeita aceitou texto dentro da imagem, mas com regras fortes:

- títulos curtos ou médios, nunca parágrafos;
- subtítulo mais leve e menor;
- hierarquia clara;
- sans-serif premium;
- alto contraste;
- espaçamento elegante;
- margens seguras de 90px;
- texto exato entre aspas;
- sem texto extra.

Prompt de tipografia recomendado:

```text
Tipografia:
Título grande, premium modern sans-serif, peso bold/semibold, alto contraste, hierarquia clara, kerning limpo, tracking levemente fechado, excelente legibilidade.
Subtítulo pequeno, sans-serif mais fino/light, elegante, com mais espaçamento e sem competir com o título.
Manter margens seguras de 90px. Não criar nenhuma palavra extra além do texto exato solicitado.
```

## Fluxo operacional no ChatGPT web

A conversa mostrou que o ChatGPT/Image-2 demora e precisa de pacing. Não bombardear prompts.

Procedimento seguro:

1. Abrir/usar a conversa ou projeto `Fluxo Refrimix Carrosséis`.
2. Primeiro pedir ao ChatGPT para criar o carrossel completo em texto:
   - título do carrossel;
   - audiência;
   - objetivo;
   - copy página por página;
   - direção visual página por página;
   - prompts individuais para Image-2;
   - legenda e hashtags;
   - checklist premium.
3. Revisar a copy ANTES de gerar imagem.
4. Quando estiver aprovado, pedir: `gere todas imagens em sequência, eu vou falando proximo`.
5. Definir explicitamente o aspect ratio: `4:5 / 1080x1350`.
6. Mandar gerar a primeira página.
7. Aguardar terminar de verdade:
   - aparecer imagem gerada;
   - aparecer botão Edit/Download;
   - parar estado de thinking/generating;
   - idealmente esperar mais alguns segundos antes de avaliar.
8. Fazer QA visual e textual da página.
9. Se aprovada, digitar exatamente: `proximo`.
10. Repetir até a página 5.
11. Não enviar `proximo` enquanto o ChatGPT ainda estiver pensando/gerando.
12. Se falhar texto/ortografia/tipografia, corrigir a página atual antes de avançar.

## Regra de decisão: corrigir ou avançar

Avançar com `proximo` somente se:
- texto está correto;
- título/subtítulo têm hierarquia clara;
- não existe texto extra;
- imagem parece premium e técnica;
- arquitetura está limpa;
- HVAC está integrado ao contexto;
- layout tem respiro;
- margens parecem seguras;
- a página cumpre sua função narrativa.

Corrigir antes de avançar se:
- título foi alterado;
- acentos ou palavras erraram;
- subtítulo ficou pesado;
- fonte ficou genérica/dura;
- imagem ficou template barata;
- apareceram máquinas soltas, técnicos stock ou clutter;
- a página ficou sem continuidade visual.

```text
Tema:
[INSERIR TEMA]

[arquitetos, engenheiros, construtoras, clientes alto padrão, hospitais, clínicas, escritórios etc.]

Objetivo:
[educar, gerar autoridade, explicar risco, vender projeto, gerar lead]

Siga o fluxo Refrimix Premium Carousel:
1. Defina título do carrossel.
2. Defina audiência.
3. Defina objetivo principal.
4. Escreva a copy das 5 páginas antes de qualquer imagem.
5. Cada página precisa ter título curto, subtítulo leve ou bullets mínimos, e uma função clara:
   - Página 1: hook forte
   - Página 2: virada de crença
   - Página 3: consequências
   - Página 4: solução técnica
   - Página 5: CTA premium
6. Crie a direção visual de cada página.
7. Crie um prompt individual de Image-2 para cada página.
8. Inclua tipografia no prompt: título bold/semibold, subtítulo light, hierarquia clara, margens seguras de 90px.
9. Revise como diretor de arte premium, copywriter de Instagram e consultor HVAC-R.
10. Entregue legenda e hashtags.

Estilo visual obrigatório:
Revista premium de arquitetura brasileira + precisão técnica HVAC-R. Off-white, cinza mineral, grafite, azul petróleo, azul claro técnico. Plantas translúcidas, linhas finas de fluxo de ar, dutos discretos, grelhas lineares, teto limpo, materiais nobres, luz natural suave, muita área de respiro.

Evite:
visual genérico de IA, template barato, excesso de ícones, texto longo, stock photo de instalador, máquina aparente sem contexto, tipografia ruim, cores demais e CTA agressivo.
```

## Prompt para iniciar geração sequencial dentro do ChatGPT

Depois de aprovar copy e prompts:

```text
Agora gere as imagens em sequência no Image-2, uma página por vez.
Use a proporção obrigatória 4:5 / 1080x1350.
Comece pela Página 1.
Depois que cada imagem terminar, eu vou digitar `proximo` para você gerar a próxima página mantendo a mesma identidade visual, paleta, tipografia e linguagem técnica premium.
Não gere a próxima página até eu escrever `proximo`.
```

## Prompt de comando curto para continuar

```text
proximo
```

O ChatGPT deve então gerar a próxima página com base na sequência já planejada. O operador Hermes precisa aguardar a geração e fazer QA antes de enviar o próximo `proximo`.

## Correções rápidas

Erro de texto:
```text
Mantenha a composição e o estilo. Corrija apenas o texto. O texto exato deve ser: "[TEXTO]". Não adicione nenhuma outra palavra.
```

Tipografia fraca:
```text
Mantenha a imagem. Melhore a tipografia para parecer editorial premium: título maior em sans-serif bold/semibold, subtítulo menor em sans-serif light, alto contraste, kerning limpo e margens de 90px.
```

Visual barato:
```text
Refaça com estética de revista premium de arquitetura brasileira, mais espaço negativo, menos ícones, menos aparência de template e mais precisão técnica sofisticada.
```

Continuidade fraca:
```text
Gere novamente mantendo a mesma paleta, tipografia, plantas translúcidas, linhas técnicas finas e atmosfera editorial das páginas anteriores.
```
