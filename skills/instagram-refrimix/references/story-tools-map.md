# Referência: Mapa Completo de Ferramentas do Story Composer

## Visão Geral

Este documento é um **mapa visual e funcional** de todas as ferramentas disponíveis no compositor de Instagram Stories. Use-o como referência rápida ao criar stories para @willrefrimix.

---

## Como Acessar o Compositor

| Método | Caminho |
|--------|---------|
| Câmera do Story | Tocar ícone câmera superior esquerda |
| Swipe | Swipe da tela principal para direita |
| Novo Post | ➕ → "Story" no seletor |
| Atalhos | Segurar ➕ e arrastar para "Story" |

---

## 1. Ferramenta de Câmera

| Função | Descrição |
|--------|-----------|
| **Foto** | Tocar para capturar imagem estática |
| **Vídeo** | Segurar para gravar (máx 60s) |
| **Boomerang** | Loop back-and-forth |
| **Rewind** | Vídeo reproduzido ao contrário |
| **Foco** | Toque para definir ponto de foco |
| **Velocidade** | Ajustar velocidade do vídeo (0.3x a 3x) |
| **Timer** | Countdown antes de capturar |
| **Flash** | Ligar/desligar flash |

---

## 2. Seletor de Galeria

| Função | Descrição |
|--------|-----------|
| **Recent** | Últimas fotos/vídeos |
| **Select Multiple** | Múltiplas mídias (para carrossel interno) |
| **Video Trim** | Cortar início/fim do vídeo |
| **Cover** | Escolher frame de capa |
| **Alignment** | Alinhar com conteúdo anterior |

---

## 3. Ferramenta de Texto [Aa]

| Recurso | Detalhes |
|---------|----------|
| **Fontes disponíveis** | Padrão, Negrito, Casual, Conhecido, Elegante |
| **Tamanhos** | Use a pinça para redimensionar |
| **Cores** | Paleta + Conta-gotas (amostragem de cor) |
| **Alinhamento** | Esquerda, Centro, Direita |
| **Animação** | Appear, Zoom, Elastic, Fade, Typewriter |
| **Fundo** | Cor sólida, gradiente, Blur (desfocado) |

### Boas Práticas @willrefrimix
- Use **Negrito** para alertas e dicas principais
- Use **Padrão** para textos explicativos
- Mantenha **máximo 6-8 palavras** por texto
- Contraste alto: fundo escuro + texto claro (ou vice-versa)

---

## 4. Ferramenta de Desenho [✏️]

| Pincel | Característica Visual |
|--------|----------------------|
| **Marcador** | Traço grosso, cor sólida |
| **Caneta** | Traço fino, alta precisão |
| **Neon** | Efeito brilhante/fluorescente |
| **Eraser** | Apaga partes do desenho |

| Cor | Padrão: branco, preto + paleta completa |
|-----|----------------------------------------|
| Opacidade | Arrastar para ajustar (0-100%) |
| Espessura | Arrastar para ajustar |

**Uso para @willrefrimix:** Circulos em fotos técnicas, setas, sublinhar pontos importantes.

---

## 5. Filtros e Efeitos

| Ferramenta | Descrição |
|------------|-----------|
| **Filtros de imagem** | Scrool horizontal: Normal, Aumentar, Vivid, Warm, Cool, etc. |
| **Face effects** | Efeitos faciais (óculos, ears, etc.) |
| **Beauty** | Suavização de pele (usar com moderação) |
| **Edit** | Ajustes manuais: Brilho, Contraste, Saturação, Vibrância, Nitidez, Sombras, Temperatura |

### Ajustes Manuais (ícone de escada)
| Ajuste | Função |
|--------|--------|
| **Brilho** | Luminosidade geral |
| **Contraste** | Diferença entre claros e escuros |
| **Saturação** | Intensidade das cores |
| **Vibrância** | Intensifica cores menos saturadas |
| ** Nitidez** | Detalhes e bordas |
| **Temperatura** | Tom quente (laranja) ou frio (azul) |

## 6. Sticker de Música [🎵]

| Recurso | Detalhe |
|---------|---------|
| **Busca** | Pesquisar por música ou artista |
| **Trecho** | Escolher 5 a 90 segundos |
| **Visualização** | Nome da música + capa do álbum |
| **Estilos de capa** | Retrato, Quadrado, Círculo |
| **Letras** | Lyrics sincronizadas (quando disponível) |

### Tipos de Exibição
| Modo | Comportamento |
|------|---------------|
| **Animado** | Texto aparece na tela em sincronia |
| **Estático** | Capa da música + nome sem animação |

**Uso para @willrefrimix:** Trilhas para vídeos de bastidores, músicas motivacionais.

---

## 7. Sticker de Enquete [📊]

| Recurso | Detalhe |
|---------|---------|
| **Pergunta** | Campo editável (até 100 caracteres) |
| **Opções** | exatamente 2 opções |
| **Duração** | 24h (padrão) ou Highlight |
| **Estilo** | Padrão, Quadrado, Bolha |
| **Cor** | Várias opções |
| **Posição** | Arrastar para qualquer lugar |

### Como Criar
1. Toque no sticker de enquete
2. Digite a pergunta
3. Digite opção 1
4. Digite opção 2
5. Ajuste estilo e posição
6. Toque para confirmar

**Uso para @willrefrimix:** Pesquisa de público, engajamento, preferências.

---

## 8. Sticker de Pergunta [❓]

| Recurso | Detalhe |
|---------|---------|
| **Tipo** | Anônima ou identificada |
| **Texto padrão** | "Faça uma pergunta" (editável) |
| **Placeholder** | "Mande sua pergunta..." |
| **Estilo** | Várias opções de bolha |
| **Respostas** | Vizíveis no story de resposta |

### Como Usar
1. Toque no sticker
2. Escolha "Anônima" ou "Identificada"
3. Personalize o texto da pergunta
4. Publique
5. Responda perguntas no día seguinte (story de resposta)

**Uso para @willrefrimix:** "Mande suas dúvidas sobre ar-condicionado!"

---

## 9. Sticker de Quiz [🧠]

| Recurso | Detalhe |
|---------|---------|
| **Pergunta** | 1 pergunta por sticker |
| **Alternativas** | 2 a 4 opções |
| **Resposta correta** | Definida pelo criador |
| **Feedback** | "Correto!" ou "Incorreto" automático |
| **Resultado** | Mostra % de acertos ao final |

### Como Criar
1. Toque no sticker Quiz
2. Digite a pergunta
3. Adicione alternativas (2-4)
4. Toque na alternativa correta
5. Escolha estilo da bolha
6. Publique

**Uso para @willrefrimix:** "Quanto tempo dura um filtro?" / "Qual consumo de um ar 12.000 BTUs?"

## 13. Sticker de Menção [@]

|---------|---------|

## Resumo Rápido — Quando Usar Cada Ferramenta

| Objetivo | Ferramenta Ideal |
|----------|------------------|
| **Educar** | Texto, Carrossel interno, Quiz |
| **Engajar** | Enquete, Pergunta, Emoji Slider |
| **Prova Social** | Foto/vídeo + texto, Menção |
| **Converter** | Link, Contagem regressiva |
| **Bastidores** | Foto/vídeo, Desenho, Música |
| **Dica prática** | Texto, Lista, Carrossel |
| **Urgência** | Contagem, Link |
| **Localização** | Sticker de Local |

## Checklist de Ferramentas por Tipo de Story

### Story de Educação
- [ ] Texto com dica principal
- [ ] Carrossel interno (se 3+ pontos)
- [ ] Hashtag relevante

### Story de Engajamento
- [ ] Enquete OU Pergunta OU Quiz
- [ ] Texto chamando ação
- [ ] Fundo visual

### Story de Conversão (Right Hook)
- [ ] Link (se disponível)
- [ ] Contagem regressiva (se promoção)
- [ ] Texto com oferta clara
- [ ] Urgência

### Story de Bastidores
- [ ] Foto/vídeo da equipe
- [ ] Música de fundo
- [ ] Texto descritivo
- [ ] Desenho (círculos, setas)
