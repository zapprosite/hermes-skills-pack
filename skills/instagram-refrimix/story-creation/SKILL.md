# Skill: Story Creation Workflow — @willrefrimix

## Propósito

Guia completo para criar Instagram Stories profissionalmente usando o compositor nativo do Instagram. Cobre acesso, ferramentas, formatação, zonas seguras e workflow de publicação para a conta @willrefrimix.

---

## Como Acessar o Compositor de Stories

### Método 1 — Ícone de Câmera
1. Abra o app Instagram
2. Toque no **ícone de câmera** no canto superior esquerdo da tela inicial
3. O compositor abre imediatamente em modo câmera

### Método 2 — Botão de Novo Post
1. Toque no ícone **+ (mais)** na barra de navegação inferior
2. Selecione **"Story"** no seletor que aparece acima das opções de post
3. Escolha foto/vídeo da galeria ou crie novo

### Método 3 — Atalhos Rápidos
- **Swipe** da tela inicial para a direita — abre câmera de stories diretamente
- Segure o ícone **+** na barra inferior e arraste para **"Story"**

## Formato e Dimensões

### Formato Instagram Story
| Medida | Valor |
|--------|-------|
| Proporção | 9:16 |
| Resolução | 1080 × 1920 pixels |
| DPI | 1080 |
| Duração padrão | 15 segundos (vídeo) |
| Duração máxima | 60 segundos (múltiplos clips) |
| Formato de arquivo | JPG, PNG (fotos) / MP4 (vídeos) |

### Formato Carrossel / Capa de Carrossel
| Medida | Valor |
|--------|-------|
| Proporção | 4:5 |
| Resolução | 1080 × 1350 pixels |
| DPI | 1080 |
| Formato de arquivo | JPG, PNG |

### Zonas Seguras — Onde Colocar Elementos

```
┌─────────────────────────────┐
│                             │  ← ZONA INSEGURA (notch/câmera frontal)
│  ┌───────────────────────┐  │
│  │                       │  │
│  │   ZONA SEGURA         │  │  ← NÃO coloca texto/CTA aqui
│  │   (conteúdo vídeo)    │  │
│  │                       │  │
│  │  ← MARGEM LATERAL →   │  │  ← Mínimo 100px das bordas
│  │                       │  │
│  │   TEXTOS/CTA          │  │  ← ZONA CENTRAL IDEAL
│  │                       │  │
│  │   STICKERS/TOOLS      │  │
│  │                       │  │
│  └───────────────────────┘  │
│         [ Swipe up ]        │  ← ZONA DE CTA (inferior, quando aplicável)
└─────────────────────────────┘
```

**Margens seguras:**
- Bordas superior/inferior: mínimo **200px** de margem
- Bordas laterais: mínimo **100px** de margem
- Zona central (400px a 1500px verticalmente): ideal para textos e CTAs
- Zona de link/swipe up: últimos **250px** inferiores

## Ferramentas do Compositor

### 1. Ferramenta de Texto [Aa]
**Função:** Adicionar textos sobrepostos ao story

| Opção | Descrição |
|-------|-----------|
| Fonte Padrão | Texto branco com sombra — mais legível |
| Fonte Negrito | Texto bold — bom para destaques |
| Fonte Casual | Estilo marcador de quadro branco |
| Fonte Conhecido | Estilo citation — elegante |
| Cores | Paleta de cores + conta-gotas |
| Alinhamento | Esquerda, centro, direita |
| Animação de texto | Zoom, aparecimento, elastisc, fade |

**Boas práticas para @willrefrimix:**
- Usar fonte **Negrito** para dicas e alertas
- Usar **Padrão** para textos explicativos
- Manter no máximo **6 palavras** por slide
- Cor de fundo escuro com texto claro funciona melhor

---

### 2. Sticker de Enquete [📊]
**Função:** Criar enquetes interativas para engajamento

| Configuração | Detalhe |
|--------------|---------|
| Pergunta | Campo editável — até 100 caracteres |
| Opções | 2 opções obrigatórias |
| Duração | 24h ou destaque permanente |
| Posição | Arrastar para qualquer lugar do story |

**Exemplo para @willrefrimix:**
- "Qual problema você mais enfrenta com ar-condicionado?"
- Opção 1: "Não gela direito"
- Opção 2: "Barulho excessivo"

---

### 3. Sticker de Pergunta [❓]
**Função:** Receber perguntas abertas dos seguidores

| Configuração | Detalhe |
|--------------|---------|
| Tipo | Pergunta anônima ou identificada |
| Texto padrão | "Faça uma pergunta" (editável) |
| Respostas | Vizíveis no story de resposta |

**Exemplo:** "Mande sua dúvida sobre ar-condicionado que respondemos no próximo story!"

---

### 4. Sticker de Quiz [🧠]
**Função:** Teste de conhecimento interativo

> ⚠️ **ALERTA 2026 — Quiz Sticker Instável:**
> Desde março 2026, o Quiz sticker foi **removido do Stories** em diversas contas (provavelmente por gating regional ou decisão da Meta). Pode não aparecer na sua bandeja de stickers.
> **Workaround:** Criar um Reel draft → adicionar quiz sticker nele → salvar/descartar → o sticker aparece em "Usados recentemente" no Stories.
> **Alternativa segura:** Use **Enquete** ou **Caixa de Pergunta** para engajamento interativo — são estáveis.

|| Configuração | Detalhe |
|--------------|---------|
| Número de perguntas | 1 por sticker |
| Alternativas | 2 a 4 opções |
| Resposta correta | Definida por você |
| Feedback | "Correto!" ou "Incorreto" automático |

**Exemplo:** "Quanto tempo dura um filtro de ar-condicionado?" / 1 mês / 3 meses / 1 ano / 2 anos

## Workflow de Criação — Passo a Passo

### Preparação
1. Defina qual slot da sequência você está criando (ver stories-jab-jab/SKILL.md)
2. Prepare a mídia (foto ou vídeo em 9:16 1080×1920)
3. Tenha o copy pronto para o slot específico

### Criação no Compositor
1. **Acesse** o compositor (câmera ou novo post → Story)
2. **Carregue** a mídia da galeria ou grave direto
3. **Aplique** filtro/ajuste se necessário
4. **Adicione** elementos na ordem:
   - Primeiro: conteúdo principal (foto/vídeo)
   - Segundo: texto (ferramenta de texto)
   - Terceiro: sticker interativo (se aplicável)
   - Quarto: stickers secundários (local, hashtag, mention)
5. **Revise** — verifique zonas seguras
6. **Publique** ou salve em Rascunho

### Após Publicar
1. **Visualize** o story como se fosse um seguidor
2. **Verifique** que todos os elementos estão legíveis
3. **Adicione aos destaques** (manter o story salvo após 24h)
4. **Monitore** engajamento nas primeiras 2 horas

## Como Adicionar Story aos Destaques

### Método Rápido
1. Quando o story expira (após 24h), ele vai para o arquivo
2. Vá em **Arquivo** → encontre o story → toque em **"Destacar"**
3. Selecione destaque existente ou crie novo
4. Escolha capa e nome do destaque

### Método Imediato
1. While viewing your own story after posting, tap the coração **"Destacar"** no canto inferior
2. Selecione ou crie um destaque

### Organização de Destaques Sugerida para @willrefrimix

| Destaque | Conteúdo |
|----------|----------|
| 📦 Serviços | Tipos de serviços oferecidos |
| 🔧 Dicas | Dicas de manutenção |
| 👷 Equipe | Apresentação da equipe |
| ⭐ Depoimentos | Clientes satisfeitos |
| 📅 Promoções | Ofertas e descontos |
| 📍 Área de Atuação | Locais atendidos |

## Arquivo de Stories

### Acessar o Arquivo
1. Vá no seu **perfil** → toque no **menu ☰** → **Arquivo**
2. Todos os stories expirets ficam aqui por 90 dias
3. Você pode re-publicar stories do arquivo

### Re-publicar do Arquivo
1. Abra o **Arquivo**
2. Toque no story desejado
3. Toque em **"..."** → **"Compartilhar story"**
4. O story é republicado com todos os elementos originais
